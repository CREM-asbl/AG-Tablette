## Install
$ npm install

## Viewing Your Application
$ npm start

## Building Your Application
$ npm build
This will create builds of your application in the dist/ directory, optimized to be served in production.

$ npm start:build
this will serve the build version in localhost

## Running Tests
$ npm test
